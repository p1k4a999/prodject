# PRD: Фриланс Воронка - Лендинг для сбора лидов

## Дата создания: 2026-02-22

## Оригинальное ТЗ
Создание landing page воронки для ниши "фриланс/удалённая работа без опыта":
- Hero секция с прогревом
- Блок проблем (почему люди не стартуют)
- Блок преимуществ (что получит пользователь)
- Социальное доказательство (отзывы, цифры)
- Lead форма (имя, email, телефон)
- Thank You Page с upsell курса $50
- Психология доверия, минималистичный дизайн

## Целевая аудитория
- Русскоязычные пользователи 20-45 лет
- Новички без опыта во фрилансе
- Люди ищущие дополнительный/удалённый заработок

## Технический стек
- Frontend: React + Tailwind CSS
- Backend: FastAPI + Python
- Database: MongoDB
- UI: Минималистичный доверительный дизайн (Playfair Display + Manrope)
